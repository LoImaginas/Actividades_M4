# Actividades_M4

Repositorio en línea para trabajar actividades de JavaScript utilizando iPad.

## Entorno de trabajo
- Editor: VS Code para iPad
- Ejecución: Navegador web (Safari / Chrome)

## Consideraciones
- Las actividades que requieren interacción con el usuario (`prompt`, `alert`) se ejecutan en el navegador.
- En el entorno de VS Code para iPad no es posible ejecutar scripts con interacción directa, por lo que el código JavaScript se incluye dentro de `index.html`.
- La consola de VS Code permite visualizar salidas con `console.log`, pero no admite entrada de datos del usuario.

## Pruebas
- Los archivos `.js` se utilizan para escritura y organización del código.
- Para pruebas con interacción, el código se ejecuta desde `index.html`.

## Antecedentes

Repositorio utilizado como ambiente de pruebas.  
Se trabajan múltiples ejercicios en un mismo repositorio, manteniendo códigos anteriores comentados para evitar la creación de múltiples repositorios de trabajo en línea.
