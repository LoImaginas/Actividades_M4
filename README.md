# Actividades_M4

Repositorio que contiene las actividades y tareas del Módulo 4, relacionadas con los fundamentos de programación en JavaScript.

Cada actividad se organiza en carpetas independientes y se prueba directamente en el navegador, utilizando index.html y archivos .js para desarrollar la lógica.

Este repositorio refleja el proceso de aprendizaje y práctica de JavaScript a lo largo del módulo.
